# كناري KSA - سوق المملكة الأول للكناري

سوق إلكتروني متكامل مخصص بالكامل للكناري في المملكة العربية السعودية.

## المميزات

- **شراء وبيع الكناري** - إعلانات موثقة مع فلترة بالمدينة والسلالة
- **المزادات الحية** - مزايدة في الوقت الحقيقي على كناري استثنائية
- **بوابة الموردين** - لوحة تحكم كاملة لإدارة المنتجات والإعلانات والطلبات
- **مستلزمات الكناري** - أقفاص، بذور، فيتامينات، معدات عرض
- **خدمات متنوعة** - فحص DNA، تدريب التغريد، نقل، استشارات
- **أدوية وعلاجات** - منتجات صحية متخصصة للكناري
- **أطباء بيطريين** - متخصصون في طب الطيور
- **المدونة** - نصائح وإرشادات وأسئلة من المجتمع
- **لوحة إدارة** - مراجعة واعتماد إعلانات الموردين
- **باقات اشتراك** - مجاني، فضي، ذهبي، بلاتيني

## التقنيات

- React + Vite
- Arabic RTL + Tajawal Font
- Mobile-first responsive design
- GitHub Pages deployment

## التشغيل المحلي

```bash
npm install
npm run dev
```
